Quick Start

Install the official npm package to get started in seconds:

npm install cloudku-uploader

Basic Upload Example

import { uploadFile } from 'cloudku-uploader';
import fs from 'fs';

const buffer = fs.readFileSync('image.jpg');
const result = await uploadFile(buffer, 'image.jpg');

if (result.status === 'success') {
  console.log('Upload URL:', result.data.url);
  console.log('File size:', result.data.size);
} else {
  console.error('Upload failed:', result.message);
}

HTTP API Reference

POST /upload

Upload a file using multipart/form-data

Parameters:

  • file - The file to upload (required)
  • expire - Expiration date (optional, ISO 8601 format)

Response:

{
  "status": "success",
  "creator": "AlfiDev",
  "data": {
    "filename": "image.jpg",
    "size": "2.1 MB",
    "url": "https://cloudku.example/abc123"
  }
}

Privacy Policy

Your privacy is our absolute priority. Here's exactly what we collect and why.

What We Don't Collect

  • Personal information or user accounts
  • IP addresses or location data
  • Browsing history or tracking cookies
  • Email addresses or contact information
  • Device fingerprints or analytics

What We Store Temporarily

  • Uploaded files (encrypted and auto-deleted after 30 days)
  • File metadata (size, type, upload timestamp)
  • Unique access links (required for file retrieval)
  • Server logs for debugging (deleted after 7 days)

Content Moderation

To maintain platform safety, our automated systems will:

  • Scan uploads for malware and viruses
  • Block illegal content and spam
  • Remove adult content automatically
  • Prevent executable file uploads

All scanning is automated - no human ever sees your files.

Data Security

Your files are protected with industry-standard security measures:

  • TLS 1.3 encryption in transit
  • Secure server infrastructure
  • Regular security audits
  • Zero-knowledge architecture
